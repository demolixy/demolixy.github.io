mvn简单说明

  1. 本地仓库
  2. maven私服仓库

1. mvn install

将编译打包之后的jar，部署到本地maven仓库

2. mvn package

只是打包部署到target下面，没有部署的本地的maven仓库

3. mvn deploy

打包部署到本地maven仓库和远程的私服仓库
